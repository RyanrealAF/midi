# StemForge

A professional-grade audio stem splitting and processing utility. This tool allows users to isolate vocals, drums, bass, and other components from any audio track with precision.

## Features
- 5-stem isolation model
- Professional studio-grade UI
- Real-time signal diagnostics
- Built-in library for session management

## Getting Started
To run this project locally:
1. `npm install`
2. `npm run build`
3. Use an HTTP server to serve the `dist/` directory (e.g., `npx http-server dist/`).
